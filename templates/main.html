Welcome to the LP Reports application!

Here you can find various dashboards for bugs on Launchpad.

You can find the most actual reports in the 'Release' and 'Triage' menus above.
The reports located in 'Project reports' are not worked on for some time, but they still should work.

You need to log in if you want to see Private or Private Security bugs. If you push 'Login' button, you will be redirected to Launchpad
where you can select duration of the token you provide to the application to search private bugs for you. Note a few things:

  • Once you issue a token on the launchpad page you will have to manually return back to application by double-clicking on 'back' button in your browser.
  • Once you log in, report generation will become considerably slower: while application stores public bugs in local DB, it will have to query launchpad using your token to generate each report to get private bugs. It takes up to several minutes for some reports.
  • Right now the feature works only for reports in 'Release' and 'Triage' menu.
  • If you ever get stuck with authentication, access this page to return things to normal: http://lp-reports-staging.vm.mirantis.net/logout.

If you have questions or suggestions or need help, feel free to reach us via lp-reports@mirantis.com mailing list.
